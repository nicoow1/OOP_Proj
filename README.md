Sprint 2: Project Description
For my project I aim to create a restaurant ordering system.
The idea is that the user will be able to through the command line build a custom order with modifiers, possibly even seeing the status of their order and checking out. I aim to hopfully have the user be able to browse the menu as well. So far, we have only learned the builder pattern, so I aim to begin the coding portion of the project by committing some OrderBuilder code that constructs an Order. The scope of the project may be subject to change as I don't know exactly what will be possible as we continue to learn more design patterns.
